# Desktop-Organizer

_Ich reinige gerne mein Windows_

**Ziel**

Üben Sie grundlegende Desktop-Navigation, das Erstellen von Dateien und Ordnern sowie das Auffinden grundlegender Systeminformationen in Ihrer Windows-VM.

**Anleitung**

1. Erstellen Sie auf dem Desktop Ihrer Windows-VM einen neuen Ordner. Nennen Sie ihn „MyCyberSecJourney“.
2. Erstellen Sie im Ordner „MyCyberSecJourney“ einen weiteren neuen Ordner mit dem Namen „OS_Notes“.
3. Erstellen Sie im Ordner „OS_Notes“ ein neues Textdokument. Nennen Sie es „Windows_Basics.txt“.
4. Öffnen Sie „Windows_Basics.txt“ und geben Sie den folgenden Satz ein: „Der Windows-Desktop ist mein neuer digitaler Spielplatz!“ Speichern und schließen Sie die Datei.
5. Finden Sie den Namen Ihres Computers heraus. (Hinweis: Sie finden ihn möglicherweise in der App „Einstellungen“ unter „System > Info“ oder indem Sie mit der rechten Maustaste auf „Dieser PC“ klicken und „Eigenschaften“ auswählen).
6. Können Sie den Hintergrund Ihres Desktops in eine einfarbige Fläche ändern? Wenn nicht, versuchen Sie, das Hintergrundbild zu ändern.

**Einreichung**

1. Machen Sie einen Screenshot Ihres Datei-Explorer-Fensters, auf dem der Ordner „MyCyberSecJourney“ und dessen Inhalt (Ordner „OS_Notes“ und Datei „Windows_Basics.txt“) zu sehen sind.
2. Geben Sie in einer Textantwort den Namen Ihrer Windows-VM (Computername) an.
3. Machen Sie einen Screenshot Ihres Desktops, auf dem das neue Aussehen zu sehen ist, und erklären Sie, warum es nicht möglich ist, nur die Hintergrundfarbe zu ändern.
