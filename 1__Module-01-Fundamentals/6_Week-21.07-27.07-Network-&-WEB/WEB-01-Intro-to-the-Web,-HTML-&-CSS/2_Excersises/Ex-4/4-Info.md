# Shmoogle

_If you build it, they will come?_

**Goal**

Replicate Google’s website with your own twist

**Instructions**

Your goal is to have a locally running website duplicate of google, but everywhere where it says google - should say “Shmoogle”.

When you click the search button it should open an alert popup (Research how to do this, requires basic JavaScript) that says “You got Shmoogled!”

**Submission**

Screenshot of your Shmoogle and the popup, and a zip containing your website files
