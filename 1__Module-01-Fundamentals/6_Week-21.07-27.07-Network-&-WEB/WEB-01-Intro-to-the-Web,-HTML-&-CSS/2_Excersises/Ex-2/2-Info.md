# Linking Around (PR)

_Let’s link up._

**Goal**

In this exercise, you’ll continue building the foundation by adding a basic navigation bar to your existing layout.

**Instructions**

1. Inside your CyberNews Tracker project folder, open the `index.html` file created in Exercise 1.
2. In the `<header>` section of `index.html`:
    - Add a `<nav>` element to contain the navigation.
    - Inside the `<nav>`, create an unordered list (`<ul>`).
    - Add three list items (`<li>`), each containing a hyperlink (`<a>`). For now, use placeholder URLs (`href=”#”`) for the links.
3. Save the file and open it in your browser to confirm the navigation appears as expected. Clicking the links won’t do anything, as we just have a placeholder target (`#`) for now.

**Submission**

Submit the updated `index.html` file containing the new navigation structure.
