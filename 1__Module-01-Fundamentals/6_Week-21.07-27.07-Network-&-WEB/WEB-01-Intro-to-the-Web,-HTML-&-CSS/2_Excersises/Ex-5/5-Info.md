# Hint Deception

_Hover for a hint, click for a journey._

**Goal**

Create a special hyperlink that reveals one URL (like a tooltip) when hovered over, and navigates to a another URL when clicked.

**The Challenge: A Two-Faced Link**

- You need a visible link on your page.
- When the user hovers their mouse over this link, a small box or piece of text (the "hint" or "tooltip") should appear near the link, providing one URL.
- When the user clicks the link, it should navigate them to a different webpage (e.g., `https://www.example.com`).
- The hint/tooltip must be created and managed using only HTML and CSS.
- The hint should be as realistic as possible, to trick users into thinking this is the correct URL.
