# Hello, Web! (PR)

_Every project starts with a simple foundation._

**Goal**

Set up the structure of your future project — _CyberNews Tracker_, a simple web app that will later display the latest cybersecurity news. In this first exercise, you’ll create the basic HTML that will serve as the skeleton for all upcoming features.

**Instructions**

1. Create a new project folder named `cybernews_tracker`.
2. Inside it, create file `index.html`.
3. In `index.html`:
    - Set up the standard HTML structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).
    - In the `<head>`, add a `<title>`: _CyberNews Tracker_.
    - In the `<body>`, create the main layout sections using semantic tags:
        - `<header>` — for the site’s title or logo (add text: _CyberNews Tracker_).
        - `<main>` — where future content (news articles) will appear. For now, include a placeholder paragraph: _"Latest cybersecurity updates will appear here soon."_
        - `<footer>` — add simple text like _© 2025 CyberNews Tracker_.

**Submission**

Submit `index.html`.
