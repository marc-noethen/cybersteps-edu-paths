# Header Honcho

_Time to get ahead...er, inspect some headers._

**Goal**

Use `curl` to fetch a webpage and examine specific request and response headers.

**Instructions**

1. Open your terminal.
2. Use `curl` with verbosity enabled to make a GET request to `https://example.com`.
3. Examine the output carefully, differentiating between request and response headers.
4. Answer the following questions based _only_ on the output:
    - What `User-Agent` string did `curl` send?
    - What `Server` software did the server report?
    - What was the exact `Content-Type` reported by the server?
    - What was the HTTP status code and reason phrase returned?

**Submission**

Submit your answers to the four questions.
